# 📚 Online Flashcards

A simple, browser‑based flashcard app for quick studying. No backend, no setup — just open and start learning.

## 🚀 Features
- Lightweight and fast  
- Loads questions from a text file  
- Tracks your score  
- Fully client‑side  
- Easy to customize

## 🖥️ Usage
Clone the repo and open:

index.html

That’s it. Works in any modern browser.

## 🛠️ Development
Edit the HTML, CSS, or JS files directly.  
Key files:
- `main.js` — flashcard logic  
- `score.js` — scoring  
- `styles.css` — UI
